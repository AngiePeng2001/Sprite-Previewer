# Sprite-Previewer
序列帧预览工具 (Sprite Sheet Previewer)

网址：
https://angiepeng2001.github.io/Sprite-Previewer/
复制链接在浏览器打开即可直接使用

一个轻量级、功能强大的在线序列帧（Sprite Sheet）动画预览与调试工具。无需安装，上传图片即可实时查看动画效果。

🌟 项目简介

在游戏开发或 UI 设计中，序列帧图片（精灵图）通常包含多个动作帧。开发者往往需要频繁调整切分参数或帧率来确认动画的流畅度。本项目提供了一个直观的界面，帮助你：

快速验证序列帧图片的切分是否准确。

实时调整播放速度（FPS）。

逐帧观察细节。

部署在自己的服务器或 GitHub Pages 供团队使用。

✨ 主要功能

🚀 即时预览：支持本地图片上传，数据完全在浏览器本地处理，隐私安全。

📐 两种切分模式：

网格切分：按指定的列数和行数均分图片。

固定尺寸：按指定的像素宽度和高度切分图片。

⏱️ 播放控制：

动态调整 FPS (帧率)（1-60 FPS）。

支持播放/暂停控制。

支持 单帧步进（上一步/下一步），方便调试关键帧。

🔍 视觉辅助：

切分网格叠加：在原图上显示半透明网格，确认对齐情况。

当前帧高亮：在原图区域实时标注当前正在播放的帧位置。

📱 响应式设计：适配 PC 和移动端，支持像素点插值优化（Pixelated Rendering），确保像素画素材不模糊。

🛠️ 技术实现

核心：HTML5 Canvas API (用于图像切分与动画渲染)。

样式：Tailwind CSS (响应式布局与现代 UI 交互)。

逻辑：原生 JavaScript (使用 requestAnimationFrame 驱动高性能动画循环)。

📖 使用指南

上传素材：点击“资源上传”区域，选择你的序列帧图片（.png, .jpg, .webp 等）。

配置切分：

如果图片是规整的网格，选择“网格切分”并输入行列数。

如果图片是长条状或特定尺寸，选择“固定尺寸”并输入单帧宽高。

设置总帧数：如果图片末尾有空白格子，可以手动减小“总帧数”以跳过空白。

调试动画：调整 FPS 进度条找到最合适的节奏，或者暂停后点击左右箭头查看每一帧的细节。

🌐 部署方法

方式一：直接运行

下载 sprite_previewer.html 文件，使用 Chrome、Edge 或 Firefox 浏览器直接打开即可。

方式二：GitHub Pages 部署

创建一个新的 GitHub 仓库。

将 sprite_previewer.html 重命名为 index.html 并上传。

在仓库的 Settings > Pages 中开启服务。

几分钟后即可通过 https://<your-username>.github.io/<repo-name> 访问。
